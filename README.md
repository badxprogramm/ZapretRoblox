# ZapretRoblox
Проект ZapretRoblox предоставляет эффективное решение для обхода ограничений на доступ к Roblox. Использует технологии DPI-обхода через подмену SNI и TLS-сигнатур. На момент публикации (03.12.2025) работают все компоненты: сайт, лаунчер и подключение к серверам. Включает актуальные списки IP-адресов и доменов Roblox.
