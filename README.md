# ZapretRoblox

Решение для обхода ограничений на Roblox (на момент публикации 03.12.2025)

**Все компоненты работают корректно: сайт, лаунчер и подключение к игровым серверам.**

## Описание

Этот проект предоставляет способ обхода ограничений, накладываемых на доступ к Roblox. Решение основано на технологии DPI (Deep Packet Inspection) обхода с использованием модифицированных правил фильтрации.

**На момент публикации (03.12.2025) полностью работают все компоненты:**
- Веб-сайт Roblox (авторизация, регистрация, профиль)
- Лаунчер Roblox
- Подключение к игровым серверам
- Загрузка и воспроизведение контента

## Почему это работает

Решение использует следующие подходы для обхода блокировок:

1. **SNI подмена** - замена Server Name Indication на домены Google, что позволяет обойти прямую блокировку по домену Roblox
2. **Подмена TLS ClientHello** - замена сигнатур TLS handshake пакетов на сигнатуры от Google сервисов
3. **Подмена QUIC initial packets** - аналогично для QUIC протокола
4. **IP фильтрация** - использование заранее подготовленных списков IP адресов Roblox

## Что было добавлено

### SNI Roblox
Добавлена поддержка обхода по SNI для всех доменов Roblox, включая:
- Все основные домены Roblox (roblox.com, www.roblox.com, auth.roblox.com и т.д.)
- Домены контента (rbxcdn.com и его вариации)
- Домены API (apis.roblox.com и поддомены)
- Специфические игровые и медиа домены

### Новые ipset
Добавлены актуальные списки IP адресов Roblox:
- IPv4 диапазоны Roblox
- IPv6 диапазоны Roblox
- Регулярно обновляемые списки для обеспечения стабильной работы
